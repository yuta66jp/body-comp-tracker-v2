# Security Policy

## Supported Versions

Only the latest state of the `main` branch is actively maintained.

| Version / Branch      | Supported |
|-----------------------|-----------|
| `main` (latest)       | ✅ Yes    |
| Older commits / tags  | ❌ No     |

## Reporting a Vulnerability

**Please do not open a public issue with vulnerability details.**

Use one of the following private channels instead:

- **GitHub Private Vulnerability Reporting** (preferred) — via the [Security tab](../../security/advisories/new) of this repository
- **GitHub Security Advisory** — if the above is unavailable, open a draft advisory directly

When reporting, please include as much of the following as possible:

- Steps to reproduce
- Affected file(s) or component(s)
- Potential impact or attack scenario

I will acknowledge the report within a few days when possible. Once confirmed, I will work on a fix before any public disclosure.

Thank you for helping keep this project secure.
