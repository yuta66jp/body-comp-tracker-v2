### 機能 2: 年度別仕上がり体重比較

**旧版仕様:**
- history.csv: Date / Weight / Label / TargetDate
- Tab2: days_out 軸（大会日=0、直前=-1, -2...）で全シーズンを重ねて比較
- Tab3: 全シーズンのタイムラインチャート + Season Low バーチャート（仕上がり体重の年次推移）
- "仕上がり" = 各シーズンの最小体重

**新版実装:**
1. `supabase/migrations/20260308000001_create_career_logs.sql`
   ```sql
   CREATE TABLE career_logs (
     id          BIGSERIAL PRIMARY KEY,
     log_date    DATE   NOT NULL,
     weight      FLOAT  NOT NULL,
     season      TEXT   NOT NULL,  -- "2024_TokyoNovice"
     target_date DATE   NOT NULL,  -- 大会日
     note        TEXT
   );
   ```
2. `src/lib/utils/calcSeason.ts` — days_out 計算・Season Low 算出
3. `src/components/history/DaysOutChart.tsx` — days_out 軸の複数シーズン重ね描画
4. `src/components/history/CareerChart.tsx` — 年別タイムライン
5. `src/components/history/SeasonLowChart.tsx` — 仕上がり体重バーチャート
6. `src/components/history/SeasonManager.tsx` — career_logs 入力フォーム
7. `src/app/history/page.tsx` — 全面リニューアル

**DB 変更:** `career_logs` テーブル追加（migration 必要）
